# MapTiler

SnapIter supports MapTiler, a modern mapping platform offering fast, high-quality map tiles with customizable styles and terrain data.
You can use MapTiler as an alternative for OpenStreetMap

To enable MapTiler in SnapIter, add your API key to the .env file:
```bash
SNAPITER_MAPTILER_KEY=your_maptiler_api_key
```

You could get an API key for free at [maptiler](https://www.maptiler.com/cloud/)
Do not forget to add your hostname to the allowed urls at the API key.
