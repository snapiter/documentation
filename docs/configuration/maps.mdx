# Maps

SnapIter uses **OpenStreetMap** (OSM) as the default map provider.

## MapTiler
To use MapTiler instead, create an API key at [maptiler.com](https://www.maptiler.com/) and set the following environment variable:
```
SNAPITER_MAPTILER_KEY=APIKEY
```
When this variable is defined, SnapIter will automatically use MapTiler as the map source.
