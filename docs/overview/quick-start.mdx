# Quick Start

Get Snapiter running in minutes.

## Requirements
- Docker  
- MapTiler account  
- SMTP server  

(See the [Requirements](/docs/install/requirements) page for details.)

## Set Up the Server
1. Install Docker  
2. Download the Docker Compose file:  
```bash
wget https://github.com/snapiter/demo/blob/main/docker-compose.yml
```

3. Configure environment variables using the .env.example

```bash
docker compose up -d
```


Snapiter is now running!
