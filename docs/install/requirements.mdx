# Requirements

## Core
- **Docker** – Runs all Snapiter services in isolated containers for easy setup and deployment.  
- **MapTiler** – Provides map tiles   
- **SMTP server** – Handles email communication for login codes.

## Frontend
- **Next.js** 

## Backend
- **Kotlin**  
- **PostgreSQL**

## Android App
- **Minimum SDK:** 33+
